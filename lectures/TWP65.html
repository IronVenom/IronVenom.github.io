Contar visitantes¶

  • Editar controlador default.py

def index():

  if not session.counter:

    session.counter = 1

  else:

    session.counter += 1

  return dict(msg="Python Zumbi", cont=session.counter)

Contar visitantes¶

  • Visão default/index

  <head></head>

    <h1>{{=msg}}</h1>

  </body>

</html>

  • Visitantes diferentes possuem contadores diferentes
